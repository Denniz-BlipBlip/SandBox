# SandBox

My repo for weekly projects, a place to experiment, learn new tools, and build small self contained things without worrying about keeping them polished or connected to each other.

Each folder is its own standalone project. Pick one and check it out.

## Why "SandBox"?

These aren't production projects. They're where I try things out, practice new concepts, and build small tools for myself. Expect rough edges, minimal docs inside each folder, and things that might change or break as I keep experimenting.
